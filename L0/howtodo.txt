лоигка программы следующая:
1. откуда-то (нам не важно) в nats server приходит сообщение
2. далее мы прослушиваем это сообщение используя наш сервер
3. полученные данные мы должны записать в postgresql
4. также полученные данные мы должны сохранить в кэш используя мапу
5. в случае падения сервиса восстанавливать кэш из постгреса
6. также нам необходимо валидировать данные поступающие из канала
7. поднять http сервер и выдавать данные по id из кэша
